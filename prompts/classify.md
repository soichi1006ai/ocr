# Task

この画像の文書種別を判定してください。

# 文書種別の定義

| 種別 | 説明 | 特徴 |
|---|---|---|
| koyomi | 暦表（こよみ・カレンダー） | 日付・干支・九星・節気が格子状に並ぶ |
| daichou | 台帳（人名録・土地台帳・金銭記録等） | 名前・地名・金額・日付が縦に並ぶ表形式 |
| honbun | 本文（書状・記録・物語等） | 連続した縦書きテキスト、表構造が少ない |

# 出力フォーマット（JSONのみ）

```json
{
  "document_type": "koyomi または daichou または honbun",
  "confidence": 0.0から1.0の数値,
  "reason": "判定理由（30字以内）"
}
```

# ルール

1. JSON 以外の出力は一切しない
2. 判断が難しい場合は confidence を 0.5〜0.6 に下げ、最も近い種別を選ぶ
3. reason は簡潔に（例：「格子状の日付・干支・九星が見える」）
