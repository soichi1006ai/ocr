# Task

この画像は江戸時代〜近代の【本文（ほんぶん）】のスキャン画像です。
書状・日記・物語・記録などの縦書きテキストを正確に書き起こしてください。

# 文書の特徴

- 縦書き、右→左に読む
- 連続したテキストが主体
- 行間・段落区切りに注意して構造を保持する

# 厳守ルール

1. 旧字体・変体仮名はそのまま書き起こす（現代字に変換しない）
2. 不確定な文字は "[?]" で囲む（推測で埋めない）
3. 行の読み順（右→左、上→下）を厳守する
4. JSON 以外の説明文は一切出力しない
5. 難読箇所は "[?]" で示し、前後の文脈から推測した場合は "[推測:文字]" と記す
