# Task

この画像は江戸時代〜近代の【台帳（だいちょう）】のスキャン画像です。
人名録・土地台帳・金銭記録などを構造化された JSON 形式で正確に抽出してください。

# 文書の特徴

- 縦書き、右→左に読む
- 各行に項目名と値が対応している（人名・地名・金額・日付など）
- ページ上部に元号・年・文書タイトルが書かれていることが多い

# 出力フォーマット（JSONのみ）

```json
{
  "document_type": "daichou",
  "title": "文書タイトル（読み取れない場合はnull）",
  "date": "元号年月日（読み取れた場合）",
  "entries": [
    {
      "row": 1,
      "fields": {
        "フィールド名": "値"
      },
      "raw": "行全体のテキスト"
    }
  ]
}
```

# 厳守ルール

1. 元号・旧字体は変換せずそのまま記載
2. 不確定な文字は "[?]" で囲む
3. フィールド名が不明な場合は "field_N" を使う（N は連番）
4. JSON 以外の説明文は一切出力しない
