# 厳守ルール

1. 元号・干支・旧字体は変換せずそのまま記載
2. 不確定な文字は "[?]" で囲む（推測で埋めない）
3. 干支は60組の順序を必ず守る（順序が崩れていたら誤認識を疑い再確認）
4. 1ヶ月の日数（28〜31日）と整合する数だけ days を出力
5. JSON 以外の説明文は一切出力しない
6. フィールドが読み取れない場合は null を使う
